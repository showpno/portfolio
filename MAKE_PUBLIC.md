## Quick Steps to Make Your Repository Public:

### Option 1: Via GitHub Web Interface (Easiest)

1. **Go to Repository Settings:**
   ðŸ‘‰ https://github.com/showpno/portfolio/settings

2. **Scroll to "Danger Zone"** (at the bottom of the page)

3. **Click "Change repository visibility"**

4. **Select "Make public"**

5. **Type the repository name** (`showpno/portfolio`) to confirm

6. **Click "I understand, change repository visibility"**

7. **Done!** Your repository is now public and GitHub Pages will work properly.
