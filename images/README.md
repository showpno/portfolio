# Profile Image

Please add your profile image here as `profile.jpg`

**Image Requirements:**
- File name: `profile.jpg` (or `.png`, `.jpeg`)
- Recommended size: 400x400px or larger (square format)
- The image will be automatically cropped to a circle

**Supported formats:** JPG, JPEG, PNG, WEBP

After adding the image, update the HTML files if you use a different filename or format.
